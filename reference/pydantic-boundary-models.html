Pydantic Boundary Models Reference

Model Boundary It Protects
FrameMetadata Camera capture to detection.
DetectedFace Detection to embedding extraction.
FaceEmbedding Embedding extraction to matching/storage.
EmployeeRecord Enrollment to storage.
MatchResult Matching to attendance decision.
LivenessResult Liveness checker to attendance decision.
AttendanceEvent Attendance decision to secure log storage.

Rules Used Here

  • extra="forbid": reject surprise fields.
  • frozen=True: keep boundary payloads immutable.
  • Confidence scores stay between 0.0 and 1.0.
  • Timestamps must be timezone-aware.
  • Embeddings must contain only finite numbers.
